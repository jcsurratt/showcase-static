RSS Feed Reader Streamlit App
Python-Based News Aggregator Built from Partial Specifications

As part of the CSC 221 Introduction to Problem Solving and Programming course, students develop a custom Streamlit web application that parses and displays RSS feeds of their choice. Working from partial design requirements, students must interpret specifications, implement Python code to fetch and parse XML-based RSS data, and dynamically render news content in an interactive dashboard. This assignment reinforces core programming concepts such as modular design, data parsing, API consumption, and user interface design—while giving students hands-on experience building real-world, data-driven applications.

Packet Capture Analysis Tool
Python + Scapy PCAPNG Analyzer for Cybersecurity

In ITP 270 Programming for Cybersecurity, students build a Python-based PCAPNG analyzer that processes large packet capture files to uncover what's happening on the network. Using the Scapy library, their script iterates through each packet to track top source and destination IPs, common protocols, and frequently used destination ports. Students also extract and correlate hostnames from DNS, NetBIOS, and DHCP traffic to map IP addresses to human-readable system names. The tool generates both on-screen summaries and a detailed summary.csv report, reinforcing real-world skills in traffic analysis, data aggregation, and Python scripting for defensive cybersecurity.

Jeopardy Buzzer System
Arduino-Based Lockout Controller

System Analysis and Design Capstone Project -Students engineered a hardware-based buzzer system for Jeopardy-style quiz games using Arduino microcontrollers programmed in a C/C++-derived language. The system uses an interrupt-driven lockout mechanism that disables all other buttons for five seconds when one is pressed, ensuring fair play. Students used CAT5 cabling for reliable multi-button communication and 3D-printed custom enclosures, integrating embedded programming with physical design.

Django Jeopardy Game System
Web-Based Quiz Platform with Real-Time Hardware Integration

System Analysis and Design Capstone Project -A full-stack Jeopardy-style quiz system built with the Django web framework. It includes an administrative interface that allows a game host to create and manage questions, categories, players, and scoring in real time. The system renders the game board dynamically in a browser and tracks player responses and point totals. It also integrates with custom Arduino-based buzzers, enabling hardware button presses to register live inputs into the game. The result is an interactive, network-based quiz platform that combines modern web development with physical computing.
